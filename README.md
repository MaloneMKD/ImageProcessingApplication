# Image Processing in C++ / WinUI 3
This project is a collection of fundamental image processing techniques implemented in raw C++ without external libraries, wrapped in a WinUI 3 application. It was developed as part of my formal training in image processing.

## 📖 Features
The application supports a wide range of operations:

### 🔹 Logical operations
logicAnd

logicNand

logicOr

logicXor

addition

subtraction

### 🔹 Contrast & intensity
linearContrast

histogramEqualization

calculateContrast

luminance

brightness

linearContrastSaturation

### 🔹 Filtering & enhancement
gaussFilter

smoothingFilter

sharpen

filtering (RGB channel filtering)

### 🔹 Edge detection
edgeDetect

robertFilter

prewittFilter

sobelFilter

laplacienConvo

### 🔹 Morphological operations
erosion

### 🔹 Geometric transformations
hFlip (horizontal flip)

vFlip (vertical flip)

rRotate (rotate right)

lRotate (rotate left)

scalingNN (nearest-neighbor scaling)

### 🔹 Thresholding & binarization
otsuBinarization
